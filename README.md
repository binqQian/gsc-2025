# vcfComp
Extension of gsc compression tool.

```
# 项目依赖：
    - brotli
    - zstd -- 提高压缩速度，但降低压缩率
    - xxhash
    - oneTBB
```

```
./build/gsc -i input.txt -o compressed_output.br
./build/gsc -i compressed_output.br -o decompressed_output.txt -d
./build/gsc -c input.txt decompressed_output.txt
```
